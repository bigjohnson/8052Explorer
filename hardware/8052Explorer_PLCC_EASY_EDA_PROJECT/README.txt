<p>This is a simple 8052 demo board with spi programming interface for Atmel 89s8252, 89s8253 and serial programmer for ds89c4x0.<br />You can configure it choosing the correct components following the backside instructions.</p>            
How to use：

At editor, open the document via: Top menu - File - Open - EasyEDA... , and select the json file, then open it at the editor, you can save it into a project.


如何使用：

打开编辑器，通过：顶部菜单 - 文件 - 打开 - 立创EDA... ，选择 json 文件打开在编辑器，你可以保存文档进工程里面。
